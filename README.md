## :star: Descrição 
Projeto de uma biblioteca digital que retorna resultados com base em títulos de livros ou autores. Foi criado
com o objetivo de praticar o uso de API's em JavaScript, além de aplicar conceitos de responsividade.

## :hammer_and_wrench: Tecnologias utilizadas
- <strong>HTML5</strong> semântico
- <strong>CSS3</strong> (media queries, flexbox, grid)
- <strong>JavaScript ES6+</strong> (fetch API, manipulação de eventos)

## :gear: Funcionalidades
- Consumo da API pública Open Library (openlibrary.org) e retorno de informações de obras literárias
- Exibição de resultados de forma dinâmica sem recarregar a página
- Link para cada resultado que redireciona para a página completa do livro na plataforma Open Library
- Layout responsivo (mobile-first)

## :computer: Como executar
Este é um projeto totalmente Front-End, sem a necessidade de um servidor para ser executado.
1. Clone o repositório: 
```
git clone https://github.com/raphaelsobral/biblioteca-pavilhao
```
2. Navegue até a pasta do projeto.
3. Abra o arquivo `index.html` diretamente no seu navegador.

## :open_book: Como usar o projeto
1. Digite o **título** ou nome do **autor** do livro que deseja encontrar na barra de pesquisa
2. Clique no botão **"Buscar"** ou pressione **Enter**
3. Os resultados da pesquisa (capa, título, autor e ano) serão exibidos logo abaixo
4. Clique na capa ou no título do livro para ver os detalhes completos do livro no site da Open Library
